# Enterprise & agency notes

## What Kinocut is for teams

Local-first, Apache-2.0 video execution for **AI agents and automation**, with preflight guardrails, quality checks, and Video Receipts so media can be reviewed before client delivery.

## Guarantees (core)

| Topic | Position |
| --- | --- |
| License | Apache-2.0 |
| Cost | Free core; no Kinocut credit meter |
| Telemetry | No phoning home in core product path |
| Media | Stays on your machine unless **you** send it elsewhere |
| Cloud keys | Not required for core FFmpeg tools |
| Support | Community via [GitHub issues](https://github.com/KyaniteLabs/kinocut/issues); no paid SLA claimed here |

## Not claimed

- Hosted multi-tenant SaaS editor  
- SOC2 / ISO certificate package (unless separately published)  
- Bundled FFmpeg redistribution in the pip wheel  
- Fully automatic brand-safe publish without human review  

## Security contact

## Commercial clarity

**Today:** Kinocut core is free open source.  

**Later:** Any commercial offering (if ever) will be announced explicitly and will not silently relicense past Apache-2.0 releases. Agencies may use Kinocut in client work under Apache-2.0 terms.

## Agency workflow suggestion

1. Keep client media local  
2. Agent drafts edits via Kinocut  
3. Require Video Receipt + quality checkpoint  
4. Human reviews hooks/thumbnails  
5. Deliver files + receipt to client
