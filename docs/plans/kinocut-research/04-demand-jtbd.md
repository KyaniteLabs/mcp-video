# Demand-side research (Claude agent) — key findings

- Incumbent SaaS marquee features: OpusClip ClipAnything+Virality Score+ReframeAnything (speaker-aware 9:16)+AI B-roll; Descript Underlord + "API + MCP (early access)" ON ITS FRONT PAGE; Submagic API live ($0.10-0.15/min metered); Captions.ai eye-contact but notorious export audio-desync; Riverside Magic Clips; CapCut.
- April 2026 benchmark: of 9 paid clippers, only 1 public REST API on entry tier, only 1 native MCP server. Time-to-first-clip on 90-min podcast: 4-25 min.
- TOP COMPLAINTS: (1) wrong-moment selection / no steerability — #1 recurring ("wish I could train Opus on MY content"); (2) credit-math resentment (1 credit/source-min, no rollover); (3) watermark ransom; (4) tier traps (Starter has NO editor); (5) export desync bugs; (6) subscription sprawl/tab-juggling; (7) hidden caps.
- LOCAL/PRIVATE DEMAND CONFIRMED: r/selfhosted asking verbatim for "local opus clip"; cottage industry of OSS clones (reelforge, Podcli, ViralMint) that market "plugs into Claude Code" AS THE INTERFACE; faceless-channel operators building FFmpeg+MCP+n8n stacks; regulated enterprise (TimeBolt Vault air-gapped pitch, on-prem Whisper boom).
- Pipeline anatomy (podcast→shorts): ingest → word-level transcript+diarization → highlight detection (hybrid ranking) → clips + aspect derivatives (shot-aware crop!) → captions burn-in/sidecar → LLM metadata → scheduling → QC/retries. ~6hr manual → 45min oversight; $18-45/episode.
- HUMANS STILL REQUIRED AT: hook selection/thumbnails, final curation, caption fixes on niche terms, first-15-seconds retention check, failure triage. Tooling that makes human review 90s instead of 1hr = the wedge.
- TOP 10 JTBD (ranked): 1) steerable/trainable best-moments detection; 2) speaker-aware vertical reframe; 3) reliable word-timed styled captions; 4) whole pipeline as ONE agent surface, no credits; 5) local/no-upload; 6) silence/filler/retake removal; 7) honest translation+dubbing; 8) multi-platform publish w/ spec validation; 9) reviewable auto b-roll/zoom/hooks; 10) QC GUARDRAILS AS A FEATURE (transcript-confidence gating, first-15s checks, idempotent publish).
- IMPLICATION: incumbents race to bolt MCP onto cloud SaaS; OSS demand is for exactly what mcp-video is (local, FFmpeg, no meter, agent-driven). Differentiators to build: steerable highlights, speaker-aware reframe, reliable captions, QC guardrails — one MCP surface, humans only at hook/thumbnail/final-review.
