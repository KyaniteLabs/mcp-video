# Code of Conduct

## Our Commitment

We are committed to making participation in the `mcp-video` community a harassment-free experience for everyone.

## Expected Behavior

- Be respectful and constructive.
- Assume good intent, and ask clarifying questions.
- Focus criticism on ideas and implementation, not people.
- Help newcomers succeed.

## Unacceptable Behavior

- Harassment, discrimination, or personal attacks.
- Trolling, insulting, or demeaning comments.
- Publishing private information without permission.
- Deliberate intimidation or sustained disruption.

## Scope

This Code of Conduct applies in project spaces, issues, pull requests, discussions, and community channels associated with this repository.

## Reporting

If you experience or witness unacceptable behavior, report it via the security/contact path in `SECURITY.md`.

## Enforcement

Maintainers may remove, edit, or reject comments, commits, code, issues, and other contributions that violate this Code of Conduct.
